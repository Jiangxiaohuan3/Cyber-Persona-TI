# CBTI
> Cyber Persona Type Indicator  
> 一份提供给互联网游民的赛博人格测试。

CBTI 像一台深夜联网后开始轻微发疯的扫描仪，把人丢进群聊、评论区、未读消息、抽象梗图和数字边界崩坏现场里，让你交出最真实的网络人格切片。

## 这是什么
一个关于 2026 数字赛博空间人格的趣味测试项目：
- 30 道高压互联网情境题
- 4 条赛博人格主轴
- 16 种数字幽灵原型
- 毒舌洞察 + 情绪安抚 + 生存建议

## 相关声明
本项目仅供娱乐和分享，它不具备任何心理测量学效力，也不能作为人格判断、职业建议或关系决策依据。
测评结果请勿当真，如有不符直接一读而过～
