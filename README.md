# CBTI

> Cyber Persona Type Indicator  
> 一份写给互联网游民的赛博人格归档。

CBTI 不是传统人格测试。  
它更像一台深夜联网后开始轻微发疯的扫描仪，把人丢进群聊、评论区、未读消息、抽象梗图和数字边界崩坏现场里，逼你交出最真实的网络人格切片。

## 这是什么

一个关于 2026 数字人格生态的趣味测试项目：

- 30 道高压互联网情境题
- 4 条赛博人格主轴
- 16 种数字幽灵原型
- 毒舌洞察 + 情绪安抚 + 生存建议

## 你可能会测到

- `E-eye`：群聊角落的电子监控探头
- `1vs9`：赛博纪检委 / 键盘斗士
- `LMAO`：抽象解构乐子人
- `Ping=460ms`：薛定谔的已读不回者
- `Happy Puppy`：无差别贴贴的快乐小狗

## 项目气质

- Cyber / Dreamy / Lo-fi / Neon Archive
- 原生 `HTML + CSS + JavaScript`
- 为互联网废土量身定做，不为现实人生负责
